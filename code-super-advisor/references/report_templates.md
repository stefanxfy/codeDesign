# 输出文档模板

本文件定义 code-super-advisor 三种工作模式的输出文档模板。无论哪种模式，最终交付物都是一份结构化 Markdown 文档。

---

## 使用说明

1. **模板是骨架，不是牢笼** — 根据实际审查内容增删章节，但保持「三层结构 + 路线图」的整体框架
2. **表格优先** — 尽量用表格呈现结构化信息，比长段落更易读
3. **每条建议可执行** — 避免笼统的"改善代码质量"，要具体到"将 XxxClass.method() 拆分为 methodA() 和 methodB()"
4. **标注知识来源** — 每个层次的结论都应注明基于哪本原书的哪个原则/模式/手法
5. **严重度必须标注** — 所有发现和建议都必须有 P0/P1/P2/P3 分级
